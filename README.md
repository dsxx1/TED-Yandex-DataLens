# 📊 TED Аналитика — Дашборд в Yandex DataLens

Интерактивный дашборд для анализа данных конференций TED, разработанный в Yandex DataLens.

![Dashboard Preview](dashboard_preview.png)

---

## 🎯 Цель проекта

Дашборд создан для компании, приобретшей лицензию на проведение конференций TED. Он помогает:
- изучить опыт прошедших конференций
- подобрать актуальные темы для выступлений
- найти интересных спикеров
- правильно организовать тайминг мероприятия

## 📌 Содержание дашборда

### Верхнеуровневые показатели
| Метрика | Значение |
|---|---|
| Количество конференций | 327 |
| Количество выступлений | 3 377 |
| Количество тем | 80 |
| Уникальных выступающих | 2 943 |
| Средняя длительность выступления | 13 мин |

### Блок "Выступления"
- **Топ-20 популярных тегов** — горизонтальная барчарт с количеством выступлений по каждой теме
- **Топ-10 смешных выступлений** — выступления с наибольшим количеством смеха в зале

### Блок "Конференции"
- **Топ-5 стран по количеству конференций** — лидирует США (1 841 конференция)
- **Топ-15 профессий по аплодисментам** — какие профессии вызывают наибольший отклик у зрителей

### Блок "Подробности конференций"
Таблица с детализацией по темам:
- Количество конференций, выступлений, спикеров
- Средняя длительность выступления
- Количество просмотров
- Среднее количество аплодисментов и смеха в зале

### Блок "Подробности выступлений"
Детальная таблица по каждому выступлению:
- Конференция и страна проведения
- Спикер
- Время выступления, просмотры, аплодисменты, смех
- Название выступления

```
ted.events          ted.talks           ted.speakers
────────────        ────────────        ────────────
conf_id (PK)   ←── event_id            author_id (PK)
event_name          talk_id (PK)   ──→ speaker_id
country             title               speaker_name
                    description         speaker_occupation
                    film_date           speaker_description
                    duration
                    views_count
                    main_tag
                    laughter_count
                    applause_count
                    language_sp
```

## ⚙️ Технические детали

### Тип чартов
Все чарты реализованы на **QL-запросах** (DataLens QL Charts) для поддержки параметризованной фильтрации через селекторы дашборда.

### Передача параметров из селекторов в QL-чарты
Параметры передаются через **алиасы связей** на дашборде:

| Поле датасета | Параметр QL-чарта |
|---|---|
| `country` | `selected_country` |
| `event_name` | `selected_event` |
| `main_tag` | `selected_tag` |
| `film_date` | `date_from` / `date_to` |

### Паттерн фильтрации для необязательных параметров
```sql
AND (NULLIF({{selected_country}}, '') IS NULL OR e.country = {{selected_country}})
```
Этот паттерн позволяет не применять фильтр, если селектор пустой.
